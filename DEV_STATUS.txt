GET  /compatibility/result       → 궁합 결과

POST /chat/message               → 챗봇 메시지 전송

POST /solo_destiny               → 나솔 제출

================================================================
  홈페이지 UI 구성
================================================================

1. 히어로 인사말 — 키워드(정인/노란색) 컬러 배지
2. 운세 점수 — 그라데이션 글로우 원형 링 + 등급 배지
3. 운세 그리드 — 재물/애정/건강/직업 색상 코딩
4. 행운의 색 — 실제 색상으로 텍스트 렌더링
5. 행운의 숫자 — 로또 공(Ball) 스타일 (각기 다른 색상)
6. 행운의 방위 — 나침반 그래픽 + 빨간 점 펄스 애니메이션
7. 띠 배지 — 이모지+이름+오행
8. 나와 비슷한 유명인 — 일주 기반 유명인 카드
9. 퀵메뉴 — 사주상세/운세상세/궁합/AI챗봇

================================================================
  사주 결과 페이지 UI 구성
================================================================

1. 히어로 헤더 — "나는 큰 나무 🌳 타입!" 쉬운 표현
2. 시간 보정 배너 — 서머타임/경도시차/야자시 칩
3. 사주 네 기둥 — 천간/지지 + 오행 컬러 태그
4. 오행 밸런스 — 막대 그래프 + 강함/부족 태그
5. 용신 — 아이콘 + 설명 + 실생활 팁 상자
6. 대운 타임라인 — 가로 스크롤 + 현재 대운 NOW 배지
7. 성격 분석 — 일간 속성별 친근한 설명
8. 추천 직업 — 오행별 구체적 직업 추천
9. 연애 스타일 — 오행별 연애 패턴
10. 십성 — 카드형 + 키워드 태그
11. 신살 — 이모지 카드 레이아웃
12. 2026 트렌드 — 커리어/연애 그리드 + 핵심 조언
13. 나와 비슷한 유명인 — 일주 기반 카드
14. 하단 액션 — 운세/궁합/챗봇/다시풀기
